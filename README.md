🛒 Binary Bandits - Grocery Store Simulator Welcome to the Binary Bandits Grocery Store Simulator! This project simulates the real-world experience of a running grocery store.

📚 Project Overview The Grocery Store Simulator allows users to:

🧑‍🤝‍🧑 Select a customer profile with unique attributes (e.g., name, cash balance).

🛍️ Browse and search aisles (canned goods, fresh items, frozen foods, etc.).

🛒 Add, remove, and view items in a shopping cart.

💳 Proceed to checkout, apply discounts, and complete purchases.

🧾 Manage store inventory and promotions dynamically.

✉️ Optionally sign up for a rewards program after checkout.

The simulation is designed to demonstrate realistic shopping habits, including budgeting challenges amid rising grocery costs.

🛠️ Built With Language: C#

Framework: MAUIAPP .NET

IDE: VS Code

Version Control: GitHub

🚀 Key Features Customer Service and Settings

Choose a customer profile with custom attributes.

Toggle between dark and light mode.

Manage user preferences.

Aisle & Item Searching

Browse store aisles with categorized sections.

View item details (name, UPC code, price, image).

Shopping Cart and Checkout

Add, remove, or edit items in a real-time cart.

Apply discounts, promo codes, or rewards.

Generate final price receipts.

Store Inventory Management

Admin tools to add, remove, and edit store inventory.

Inventory adjusts dynamically after each transaction.

Promotions & Discount Manager

Automated discounts like BOGO (Buy One Get One), 10% off, fixed dollar-off coupons.
